Opposing the "wrinkle removing" substances that claimed to use natural herbs to "reverse the aging process", there have been new groups arising that have instead actually been researching the healthy extension of life, with the goal of eventually suspending and even reversing the biological aging process of humans. 27th December 2013 | Comments

Why not start my blog off with a post about Artificial Intelligence? Firstly, we need to create a specific definition of intelligence. A perfectly intelligent system for n nodes is, by definition, a system that is able to best take an input, and give the quickest and best desired output, and thus has the smallest latency that is physically possible between nodes, with the least number of errors in communication, with the highest order complexity given the physical properties of the system itself, using the simplest method possible to obtain the highest desired output (or the output with the greatest coherency).
